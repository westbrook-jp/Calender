Calender : Calender on Windows Desktop

単純にカレンダーを表示するデスクトップアクセサリアプリケーションです。
Windows Vista/7ではデスクトップガジェットに使い勝手の良いものがあったのですが、
Windows 8でデスクトップガジェットの機能がなくなったため、同様の使い勝手を目指して
作っています。

■最新版置き場
  http://www.westbrook.jp/Software/Calender/

■動作環境
・.NET Framework 4.0 / WPF使用
  Windows 7などで.NET Framework 4が入っていない環境では
  Windows Updateまたは以下のURLから.NET Framework 4を
  インストールしてください。
  http://www.microsoft.com/ja-jp/download/details.aspx?id=17851
  Windows 8では標準で.NET Framework 4が使用できます。

■使用開始方法
・適当なインストールフォルダにファイルをすべて展開する。
・Westbrook.Calender.exeを起動する。

■使用法
・アプリケーション画面上で右クリックすると操作メニューが出ます。

■ログイン時に自動起動する方法
・"プログラム名を指定して実行"(Windows 8ではWindowsキー+R)。
・名前に"shell:startup"と入力しOK。
・表示されたフォルダに当アプリケーションのショートカットを作成

■注意点その他
・設定等は"%userprofile%\AppData\Local\Westbrook\"以下の
  Westbrook.Events.exe_Url_xxxxxxxxx (xxxxxxxxxの部分は環境によって変化する)に
  保存されています。
・インストールしたフォルダと、上記設定フォルダ以外は使用していません。
・複数のGoogleアカウントで使用したい場合は、別のフォルダにインストールする
  ことで可能です。
